Instruções para executar o agente

Para salvar a imagem do agente no servidor, execute o seguinte comando:

docker save -o /caminho/para/saida/imagem.tar meu-repositorio:minha-tag

Para carregar a imagem do agente em outra máquina, execute:

docker load -i ~/imagem.tar
docker run --name <AGENT_NAME> -e AGENT_NAME=<AGENT_NAME> -e CENTRAL_SERVER_URL=<CENTRAL_SERVER_IP> --network host -d devsecops-monitoring-agent:latest
